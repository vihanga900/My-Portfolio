My Projects

    Created a wearable device that monitors vital signs such as heart rate and temperature, and sends data to a cloud server for real-time analysis. This project aimed to provide remote health monitoring for patients.

    Built an IoT-based parking management system using sensors to detect available spaces and relay data to a mobile app for real-time parking availability.

    Technologies Used: ESP32 microcontroller, Sensors, RFID module, MQTT protocol, Firebase

    Created a portable air quality monitoring device using various sensors (CO2, PM2.5, temperature, and humidity) connected via IoT to a central dashboard for real-time monitoring and analysis.

    Technologies Used: ESP8266 microcontroller, Sensors, MQTT/HTTP protocol, Firebase, ThingSpeak
